Medication Reminder Program

Keep track of your medication schedule effortlessly

Why You Should Use This Application

This Medication Reminder Program is designed to help users manage their medication schedules effectively. Here are the key benefits:

  • Never forget your medication again with timely reminders.
  • Customizable interval settings to suit your needs.
  • Simple, easy-to-use, and built with Rust for performance.
  • Clear and intuitive text-based interface for all ages.
  • Manage multiple reminders with ease.

How It Works

The program lets you set reminders for any medication by specifying its name and how often you need to take it (e.g., every 30 minutes). You will receive alerts at the specified interval, ensuring you never forget your doses.

Installation and Usage

Installation

Follow these steps to install the Medication Reminder Program on your system:

  1. Clone the repository: Open your terminal and run the following command: 
    git clone https://github.com/Thess1999/medication_reminder.git
  2. Navigate to the project directory: 
    cd medication_reminder
  3. Build and run the project: Ensure that Rust is installed, then run: 
    cargo run

Usage

Once the program is running, you will be presented with a menu of options:

  1. Add Reminder – Enter a medication and set a reminder interval.
  2. List Reminders – View all your current medication reminders.
  3. Remove Reminder – Delete a reminder by providing its ID.
  4. Exit – Exit the program.
